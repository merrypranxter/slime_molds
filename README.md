# Slime Mold Visual Taxonomy & Prompt Arsenal

A structured database of slime molds (myxomycetes, dictyostelids, protostelids) designed for AI art generation, creative coding, shader experiments, and visual worldbuilding.

## What This Repo Does

This is **not** a dry biology reference. This is a **visual DNA library** for generative systems.

Every species file includes:
- Morphology & lifecycle stages
- Texture, color, surface traits
- Shape language & branching logic
- Substrate contexts
- Art translation targets (architecture, fashion, alien biology, materials)
- Render mode suggestions (macro, mutation, mathy networks)

## Why Slime Molds

Slime molds are shapeshifting organisms that pass through multiple visually distinct lifecycle stages:
- **Plasmodial slime molds** (myxomycetes): Foams, veins, coral tufts, metallic bead towers, spreading yellow blobs
- **Cellular slime molds** (dictyostelids): Coordinated amoeba swarms forming migrating slugs
- **Protostelids**: Tiny stalked minimalist freaks

They grow on bark, rotting logs, mulch, dung, moss, and occasionally in caves or deserts. They're **macro-scale weird** (visible without a microscope) and **lifecycle-dramatic** (spore → amoeba → giant multinucleate blob → fruiting body).
